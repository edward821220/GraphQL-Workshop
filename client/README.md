# GraphQL Workshop

#### Preset

- Step1: Run `bun install --frozen-lockfile` or `npm ci`

- Step2: Update the API URL.

- Step3: Run `watch` script in package.json to generate the types files for graphql api.

- Step4: Run `dev` script
